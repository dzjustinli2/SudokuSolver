Requirements:
Java Runtime Environement 7

Eclipse SDK Version: 3.7.2 for running and debugging source code.

---------------------------------------------------------------------------------------------------------------
Instructions:
Run the Sudoku.jar file to start the game.

Enter the Sudoku puzzle in the left text area as follows:

 1) Enter each row with a space between elements
 2) Enter 0 for missing element

e.g.

 5 3 0 0 7 0 0 0 0
 6 0 0 1 9 5 0 0 0
 0 9 8 0 0 0 0 6 0
 8 0 0 0 6 0 0 0 3
 4 0 0 8 0 3 0 0 1
 7 0 0 0 2 0 0 0 6
 0 6 0 0 0 0 2 8 0 
 0 0 0 4 1 9 0 0 5
 0 0 0 0 8 0 0 7 9
